# 🛠️ SMARTWARE - DIAGNÓSTICO INTELIGENTE v1.3.4 (BETA)
> **Herramienta de diagnóstico inteligente para Windows**

Gracias por confiar en mi trabajo. Este software ha sido diseñado para entregarte una visión honesta, técnica y comprensible sobre la salud de tu computadora.

---

## 🚀 1. Perfiles de Uso y Kit Pro
SmartWare se adapta a tu perfil para ofrecerte la experiencia más eficiente. Incluye un **Kit de Diagnóstico Profundo** con herramientas líderes (portables):
* **CrystalDiskInfo** (Salud de disco)
* **HWMonitor** (Temperaturas y voltajes)
* **TreeSize Free** (Mapas de almacenamiento)
* **AdwCleaner** (Seguridad y limpieza)

### 👤 ¿Cómo funciona el acceso?
* **MODO ESTÁNDAR:** Ideal para un análisis rápido. SmartWare realiza el diagnóstico por ti y genera un reporte claro en el escritorio.
* **MODO TÉCNICO/AVANZADO:** Habilita el despliegue de las herramientas de bajo nivel para visualizar datos granulares en tiempo real.

---

## 🔒 Tu tranquilidad es lo primero
-   **CONTROL TOTAL:** Nada se ejecuta sin validar primero el estrés de tu sistema para evitar bloqueos.
-   **SIN INSTALACIÓN:** Versiones portables que no ensucian tu registro ni menú de inicio.
-   **LIMPIEZA AUTOMÁTICA:** Al cerrar, el script ejecuta un protocolo que **ELIMINA** todos los temporales. Tu equipo queda limpio, solo con tu reporte en el escritorio.

---

## 📦 2. Instrucciones de Uso
1.  **EJECUCIÓN:** Haz clic derecho sobre `script.ps1` y selecciona **"Ejecutar con PowerShell"**.
2.  **PERMISOS:** Requiere privilegios de **Administrador**. Selecciona "Sí" cuando el sistema lo solicite.
3.  **SEGURIDAD:** Al ser software independiente, Windows podría mostrar un aviso de SmartScreen. Haz clic en *"Más información"* -> *"Ejecutar de todas formas"*.
4.  **REPORTE:** Al finalizar, busca el archivo `.txt` con los resultados en tu escritorio.

> [!IMPORTANT]
> En caso de errores, se generará un archivo **SmartWare_Debug.log**. Compartirme este archivo me ayuda muchísimo a mejorar la herramienta.
